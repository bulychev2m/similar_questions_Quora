## Система подсказок похожих вопросов на Quora

Этот проект представляет собой микросервис на Flask, который реализует систему подсказок похожих вопросов на данных сайта Quora. Основные компоненты системы:

1. Фильтрация по языку: Использование библиотеки LangDetect для исключения вопросов не на английском языке.
2. Поиск кандидатов: Применение FAISS для поиска вопросов-кандидатов, похожих на входной запрос, на основе векторизации с помощью предобученных GloVe эмбеддингов.
3. Ранжирование кандидатов: Использование модели KNRM для повторного ранжирования кандидатов и выбора топ-10 наиболее похожих вопросов.

Проект включает в себя реализацию двух основных эндпоинтов:

- /query: Принимает список запросов и возвращает для каждого список из 10 наиболее похожих документов.
- /update_index: Принимает словарь документов, среди которых будут искаться ближайшие к запросам.


knrm.py - реализация Kernel-based Neural Ranking Model (K-NRM) из [статьи](https://www.cs.cmu.edu/~zhuyund/papers/end-end-neural.pdf),ее обучение на датасете Quora, и оценка качества модели.
